## Food Recipe APP
 It is a web application that can search for recipes. Made this by using Free Meal API. 
 
## Technologies we used:
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

## Features:
<ul>
  <li>API Fetching</li>
  <li> Sign-in/Sign-up</li>
  <li>Display random recipie</li>
 <li>Display search input</li>
</ul>
